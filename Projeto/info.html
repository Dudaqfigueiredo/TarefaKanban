Tag O que faz?
head Onde ficará as informações de configuração. São tags que não são visíveis, tags que vão informar para o browser algumas configurações da página
body Onde inserir elementos que compõe a estrutura da página.
Form A tag é usada para criar um formulário HTML para entrada do usuário.
input A tag especifica um campo de entrada onde o usuário pode inserir dados.
button A tag define um botão clicável.
nav A tag define um conjunto de links de navegação.
table A tag define uma tabela HTML.
footer A tag define um rodapé para um documento ou seção.
div A tag define uma divisão ou uma seção em um documento HTML.
h1 - h6 As tags to são usadas para definir títulos HTML.
a A tag define um hiperlink, que é usado para vincular de uma página a outra.
center A tag foi usada em HTML4 para alinhar o texto do centro.
meta A tag define metadados sobre um documento HTML. Metadados são dados (informações) sobre dados.
